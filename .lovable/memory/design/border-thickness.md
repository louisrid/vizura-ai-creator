---
name: Border thickness
description: All borders use 2px consistently. Selection highlights use 3px yellow. No grey borders on non-selected images.
type: design
---
All borders standardised to 2px across all components and pages. Selection highlights (pick face, character grid new item) use 3px solid #facc15. Non-selected images have no border (padding used to maintain size). Yellow accent borders on character detail boxes use 2px solid rgba(250,204,21,0.25). Trait pills use 2px solid #222. Buttons match 2px borders. Toast and UI components also 2px.
